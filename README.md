# PLANA
Premier League data analysis

使用python的各种工具获取英超数据，并分析，最后展示。
1. 爬虫
2. 分析
3. 展示
4. 管理

## 爬虫
### 来源
虎扑。只获取英超排名球队数据。球队名称，场次，胜，负，平，进球，失球。剩下的数据由数据分析完成。
### 版本1
1. requests 负责请求网页数据
2. xpath 负责解析数据
3. mongodb 负责存储爬取的数据
